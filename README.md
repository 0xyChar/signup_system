A web app built with Django REST Framework and Vue.js that handles user registration, login, email verification, password reset and a personalized dashboard that displays articles.The backend uses Django with PostgreSQL for data storage and JWT for authentication,while the frontend is built with Vue.js.

A web app where users share their favourite articles.

Features
- User authentication (signup, login, email verification, password reset).
- Add article links with automatic metadata fetching.
- Articles added can be seen and read by every registered user.
